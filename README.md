## What's Inside

```
conference-portfolio-builder/
│
├── learn-guide/             ← Main learning website (Astro)
│   ├── src/
│   │   ├── layouts/         ← Layout components
│   │   └── pages/           ← Route pages (kids, adults, templates, etc.)
│   └── package.json
│
├── templates/               ← Ready-to-use project templates
│   ├── portfolio-basic/     ← Simple portfolio (single HTML)
│   ├── portfolio-ai/        ← AI-powered portfolio with chatbot
│   ├── game-template/       ← Simple browser game
│   └── chatbot/             ← AI chatbot base
│
└── tutorials/               ← Step-by-step written guides
```

## Quick Start

### Requirements
- [Ollama](https://ollama.com) (free, local AI)
- [Claude Code](https://claude.ai/code)

### Steps
1. Install Ollama
2. Get Claude Code
3. Pick a template from `templates/`
4. Open in browser and start customizing
